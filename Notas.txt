¿Que es un Framework FrontEnd o Framework CSS?

Son una base para empezar un proyecto web permitiendo flexibilidad en el diseño. 
Organizacion y estructura de nuestro HTML, CSS y JS. Posee una Grilla (Grill) como caracteristica principal, 
Estilis para las fuentes y componentes visuales pre-armados. Teniendo como meta final el ahorro de tiempo
En la programacion de un sitioweb, dandonos Componentes visuales testeados. Responsive Desing 
ademas de asegurarnos que nuestro codigo no va a posee errores.
